Como fuinciona a HTML e as CSS?

A linguagem HTML é uma linguagem focada em conteúdo (textos, imagens, vídeos, tabelas), enquanto a CSS é uma linguagem focada em design (cores, sombras, tamanhos, posicionamento), JS (JavaScript) é uma linguagem (de programação) focada em interações (menus, animações, popups, validações)

Direitos de Uso de Imagem no seu Site

Direitos autorais existem e podem dar problema, caso não haja cuidado ao selecionar os arquivos.
Indo no Google/Imagens, em ferramentas de pesquisa, observamos a aba 'Direitos de uso'. Selecionando a mesma, podemos observar as seguintes opções:

→ Sem filtro de licença - Todas as imagens da internet

→ Marcadas para reutilização com modificação - Essas podemos usar e modificar a vontade, sem problemas

→Marcadas para reutilização - Podemos usar, mas sem modificar

→ Marcadas para reutilização não comercial com modificação - Só pode ser utilizada e modificada, caso não tenha fim comercial

→ Marcadas para reutilização não comercial - Só pode ser utilizada, caso não tenha fim comercial, não poderá ser modificada em hipótese alguma

Formatos para Imagens na Web

Existem basicamente 2 tipos de arquivos para imagens que são utilizados na Web: JPEG e PNG

→ JPEG - Alta compactação de imagens (Transforma eles em arquivos mais leves, sem deixar a qualidade cair)

→ PNG - Permite transparência e animação
