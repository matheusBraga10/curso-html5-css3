Como fuinciona a HTML e as CSS?

A linguagem HTML é uma linguagem focada em conteúdo (textos, imagens, vídeos, tabelas), enquanto a CSS é uma linguagem focada em design (cores, sombras, tamanhos, posicionamento), JS (JavaScript) é uma linguagem (de programação) focada em interações (menus, animações, popups, validações)

Listas em HTML 5

Listas Ordenadas

Listas servem para simplificar e sintetizar o conteúdo a ser passado e faz a demarcação de cada item para facilitar a visualização.

Para criar uma lista ordenada, vamos usar a tag <ol> (ordered lists) para delimitar e <li> (list item) para identificar cada item.

  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5

Parâmetro Type

Listas ordenadas possuem um parâmetro type, onde configura-se o tipo de marcador da lista. As opções são:

  • 1: 1, 2, 3, 4...
  • A: A, B, C, D...
  • a: a, b, c, d...
  • I: I, II, III, IV...
  • i: i, ii, iii, iv...

Listas não Ordenadas

Para criar listas não ordenadas, vamos usar a tag <ul> (unordered lists), para delimitar a lista e a tag <li> ára criar cada um dos seus elementos.

  • type="disc"
  • type="circle"
  • type="square"

Juntando Listas

Podemos também criar listas mistas, configurando listas dentro de outras listas. Veja a seguir:


    <ol>
        <li>Item 1</li>
        <li>Item 2</li>
    <ol type="a">
        <li>Item 2.1</li>
        <li>Item 2.2</li>
        <li>Item 2.3</li>
    </ol>
        <li>Item 3</li>
        <li>Item 4</li>
        <li>Item 5</li>
    </ol>

Que resultará em:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4
  5. Item 5
